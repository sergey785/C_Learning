# Учебный проект по языку C

Этот проект содержит различные задачи и примеры кода по изучению языка C, включая темы:
- Арифметические операции
- Булевы выражения
- Логические операторы и многое другое

## Структура проекта:

- **02_arithmetic** — Примеры арифметических операций.
- **03_booleans** — Примеры работы с булевыми типами данных и логическими операторами.

## Описание файлов:

- **booleans_task.c**: Задача, которая проверяет булевы выражения на основе ввода пользователя.
- **booleans_notes.c**: Интерактивная шпаргалка по булевым выражениям с объяснениями операторов.

## Как использовать:

1. Откройте соответствующий `.c` файл.
2. Скомпилируйте и запустите программу.
3. Следуйте инструкциям на экране.
